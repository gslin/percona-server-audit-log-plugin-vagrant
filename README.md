Percona Server 5.5 with Audit Log Plugin Vagrantfile
====================================================

Overview
--------

This <code>Vagrantfile</code> will install a Percona Server 5.5 and audit log plugin.

Tested on Vagrant 1.6.2 (w/ Docker).
